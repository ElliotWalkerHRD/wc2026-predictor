Seed R32 Bracket
Computes final group standings, ranks 3rd-place teams, runs FIFA bipartite matching, and writes home/away team codes to match_results for matches 73–88. Auto-runs via cron once all 12 groups finish. Use this button to force or re-seed manually.
Seed Next Round
Reads finished knockout results and writes the advancing team into each next-round slot (R32→R16, R16→QF, QF→SF, SF→Final). Idempotent — safe to run multiple times. Auto-runs every 2 min via cron; use this to force immediately after a match finishes.
To disable the cron job entirely (vs. just toggling the kill-switch): run UPDATE cron.job SET active = FALSE WHERE jobname = 'wc2026-auto-update'; in the SQL Editor.

Round Lock Overrides

Override the time-based lock for any round. Force Open keeps predictions editable even after the lock time. Force Locked closes a round early. Normal removes the override and restores time-based locking.
